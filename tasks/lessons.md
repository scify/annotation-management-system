# Lessons

## Figma mockups are the source of truth for visuals
- 2026-06-10, Co-Managers tab wiring: the task description said the remove button should be "like in the annotators tab" (icon button), but the Figma mockup showed a labeled blue "Remove" button — the user ultimately chose the Figma version.
- Rule: when a verbal description and the linked Figma mockup disagree on visual details, default to the Figma mockup; only ask if the difference is behavioural.
