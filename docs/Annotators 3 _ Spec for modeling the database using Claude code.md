

# What this project is all about

We are working on an application that will allow annotators to login and complete annotation tasks. This application enables annotation management.  
An admin can create a project, describe the annotation task and assign annotators.  
Then the annotators can login and start the annotation tasks and datasets. This application allows admins to assign certain annotation tasks to annotation managers and grant access to certain annotators. The annotation managers can create Projects about specific annotation tasks and datasets and then initiate an annotation subproject about a subset of the dataset that will be assigned to specific annotators. The Project manager can share the Project to other Project managers, although only one manager is considered to be the owner. The annotators can select a subproject assigned to them and work on that by performing annotations. The Project  manager can monitor the progress of all Project and Subprojects and view notification messages that are sent to him by annotators and other annotation managers.

Each annotation task is custom and we do not know a priori the elements that compose it. In general each dataset contains dataset instances and each dataset instance contains several dataset elements in the form of key, value, index elements. In the same way an annotation about a specific Project and dataset is composed by several annotation elements (key, value, index). Each dataset instance in a Project can be annotated by several annotators.

At the end of an annotation SubProject the annotation manager should be able to extract the annotations regarding this Project.
